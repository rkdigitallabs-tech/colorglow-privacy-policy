==================================================================
TROUBLESHOOTING
==================================================================

If build fails:

1. Check Java version: java -version (should be 17)
2. Check Gradle version: ./gradlew --version
3. Sync project: File > Sync Project with Gradle Files
4. Invalidate caches: File > Invalidate Caches / Restart
5. Check Android SDK path in local.properties

Common errors:
- "SDK location not found" → Create local.properties
- "Unsupported class file version" → Update Java to 17
- "Failed to resolve" → Check internet connection
